# School ERP System

A full-stack School ERP System developed as a Software Engineering project.

## Features

- Authentication & Authorization
- User Management
- Student Management
- Teacher Management
- Attendance Management
- Timetable Management
- Examination Management
- Dashboard & Reports

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
